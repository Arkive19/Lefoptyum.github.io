1970's "Hippie" era - New Road, Kathmandu

Project Type: Passion Project
Description: This passion project is a historically inspired digital recreation of New Road, the economic heart of Kathmandu, during the iconic hippie era of the 1960s. Since this is only a showcase, it is not a full experience and it is meant to act as a picturesque recreation of my home town.

Highlands (HALO Military Simulation)

Project Type: Commission
Description: A paid commission to create a futuristic military simulation inspired by the Halo universe, set on a mountain landscape. The environment features dark metal, purpose-built and functional buildings, all interconnected to support strategic movement and gameplay. A central river runs through the terrain and flows into a dam, adding both visual depth and utility. Designed with a strong focus on sci-fi realism and immersive detail, the project met all client expectations and was delivered to full satisfaction.

WW2 Forest Deployment Map

Project Type: Passion Project
Description: A detailed environment inspired by German World War II deployment maps, set in a dense forest with tall trees and heavy jungle foliage. The map features authentic German-style WW2 houses, strategic camps, and interconnected roads designed for tactical realism. Created as a passion project to explore historical accuracy, atmosphere, and immersive level design.
